# Dictionary of Lèvynja
![GitHub last commit](https://img.shields.io/github/last-commit/Academy-of-Levynja/dictionary?style=for-the-badge)
![GitHub repo size](https://img.shields.io/github/repo-size/Academy-of-Levynja/dictionary?style=for-the-badge)

The dictionary of Lèvynja is the central and official dictionary for the constructed language. It is written in JSON, which makes it readable by both machines and humans. It is licensed under CC0, placing it in the public domain because it is a constructed language, and Lèvynja as a language should be free for use.
